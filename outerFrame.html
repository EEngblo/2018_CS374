그래픽 카드

그래픽 카드는 게임 화면을 모니터에 그려내는 부품입니다.

아래의 프레임이 높은 제품을 사용할수록
더 부드럽고 사실적인 게임 화면을 볼 수 있습니다.

제품정보 프레임
 성능
 가격
CPU

CPU는 그래픽 카드가 하는 일인 화면을 그리는 것을 제외한 모든 작업을 수행하는 부품입니다.

좋은 부품을 사용하면 전반적인 컴퓨터 성능이 향상됩니다.

제품정보 프레임
 성능
 가격
RAM
상세정보

컴퓨터가 실행중인 프로그램에 대한 정보를 적어두기 위해 사용하는 메모지 같은 역할을 합니다.

게임 성능에는 큰 영향을 주지 않지만, 가득 찰 경우 급격히 느려지고 넘치게 되면 게임이 팅길 수 있습니다.

용량이 클수록 동시에 더 많은 게임과 프로그램들을 실행할 수 있습니다.
저장장치
상세정보

사용자 데이터를 저장하는 공간으로, 초고속 저장장치인 SSD와 하드디스크가 있습니다.

빠르지만 작고 비싼 SSD에는 윈도우, 자주 사용하는 게임과 프로그램들을 저장하여 빠르게 실행하고,

용량이 크지만 느린 하드디스크에는 가끔 사용하는 게임, 프로그램, 음악, 동영상, 사진 등을 저장하시면 됩니다.


현재 사용중인 컴퓨터의 저장장치 용량, 주로 하는 게임의 개수와 용량을 확인해 보세요.

그래픽 카드 성능

그래픽 카드 성능의 발목을 잡지

않을 CPU를 썼다는 가정하에 같은

프로그램을 돌릴때 얼마만큼의

성능을 낼 수 있는지에 대한 지표

입니다.

수치가 높을 수록 좋은 그래픽

카드 입니다.

CPU 성능

Cinebench라는 CPU 성능 측정

프로그램을 통해 측정된 CPU의

성능지표 입니다. 다른 부품은

CPU 성능의 발목을 잡지 않는

선에서 모두 같은 걸 썼다고 가정

하고 측정된 가상의 성능입니다.

수치가 높을 수록 좋은 CPU입니다.
